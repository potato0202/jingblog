Hexo部落格圖片放哪邊? 免費圖床服務介紹

發表於2024-06-17|更新於2024-06-17|Hexo
|閱讀量:

可分成 放在 Github ( 就是跟網站檔案放一起 ) 和 firebase 圖床服務

(一) 在 source 資料夾右鍵新增 images 資料夾

在部落格文章添加圖片方式，MD寫法 ▼
![alt名稱](/images/圖片名稱.png)

GitHub 使用限制
您可以將網站託管在 GitHub 的 github.io 或您自己的自訂域上。

GitHub Pages 網站受以下使用限制的約束：

  • GitHub Pages 儲存庫的建議限制為 1 GB。
    已發佈的 GitHub Pages 網站不得大於 1 GB。

  • 頻寬每月100GB ( 可是老師說沒被限過 )

(二) firebase 圖床服務

→ 官網連結
Google的平台，有提供 5G 容量

  • 進入官網登入後，點右上角 go to console，點擊”創建項目”，開始取名字等等操作。

項目成立後，左側 構建 >> Storage 點進去 >> 開始啟用 >> 以正式版模式 >> Cloud Storage 位置 本來想選asia-east1(台灣)但是出現警告類文字，就選name5 >> 等他跑完 >> 可以開始上傳檔案了。

在部落格文章添加圖片方式，MD寫法 ▼
